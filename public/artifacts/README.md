# Imani Logo and Favicon Artifacts

These files are generated from the official supplied Imani vector files:

- `imani-mark.svg`
- `imani-logo.svg`
- `imani-logotype-dark.svg`
- `imani-logotype-light.svg`
- `/favicon-real.svg`
- `/imani-logotype.svg`
- `/imani-logotype-light.svg`
- `/favicon.svg`
- `/icons/*`
- `/og/*`

Run `bun run assets:brand` after changing source SVGs. Keep the approved color treatments:

- White or Cloud Bleach background: Tunic Green mark, Fence Green wordmark.
- Tunic Green background: White mark, Fence Green wordmark.
- Fence Green background: Tunic Green mark, White wordmark.
